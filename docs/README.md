# HOWTO build an emulator with zero coding skills!

1. ***Open*** your favourite LLM/AI such as ***<a href="https://m365.cloud.microsoft/chat/?auth=2" target="_blank">Copilot</a>***, ChatGPT or <a href="https://gemini.google.com" target="_blank">gemini</a>.
2. ***Copy paste*** the full text of <a href="./02 Technische specs.md" alt="Technische specs">02 Technische specs.md</a> to a LLM chat window.
3. Press enter.  Wait 1 minute for it to generate code.
4. ***Confirm*** of type in : "build the sn76489_emulator.py immediatley, for me to test on my laptop"
5. ***Download*** the genereated code
6. Install the latest <a href="https://python.org">Python</a> for free.
7. ***Run*** the emulator code (Python must be installed!)

CONGRATS!!!  You developed a python app without any Python / coding skills whatsoever :)
